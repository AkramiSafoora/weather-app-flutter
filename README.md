# Flutter Weather App

A simple Flutter weather application using the OpenWeather API.

## Features

- Search weather by city
- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- Weather condition

## Getting Started

1. Get a free API key from OpenWeather.
2. Open:

lib/services/weather_service.dart

3. Replace:

```dart
YOUR_API_KEY_HERE
```

with your own API key.

4. Run the project.
